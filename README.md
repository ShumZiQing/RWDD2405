# EcoConnect
**EcoConnect** is a comprehensive web-based platform dedicated to promoting environmental sustainability and eco-conscious living. Built with PHP, CSS, and JavaScript, this application serves as a community hub for sharing, learning, and collaborating on eco-friendly initiatives.

## Overview
EcoConnect connects users passionate about environmental conservation through an intuitive platform that facilitates knowledge sharing about energy conservation, sustainable gardening practices, and community environmental projects. The platform supports both regular users and administrators in managing activities, collaborators, and sustainability resources.

## Key Features
### For Users
- **User Authentication**: Secure login and registration system with role-based access control
- **Energy Conservation Tips**: Browse and interact with curated energy-saving tips for home and workplace
- **Gardening Tips**: Access sustainable gardening practices and eco-friendly gardening advice
- **Community Projects**: View and participate in environmental projects and community programs
- **Business Directory**: Discover eco-conscious businesses in your area
- **Interactive Engagement**: Like tips and favorite businesses to build a personalized collection
- **Responsive Design**: Fully optimized for desktop and mobile devices

### For Administrators
- **User Management**: Oversee user accounts and manage user access
- **Content Management**: Add and edit energy conservation tips and gardening guides
- **Activity Management**: Create, manage, and delete community projects and programs
- **Collaborator Management**: Manage and coordinate with environmental partners and collaborators
- **Business Oversight**: Monitor and manage partnered eco-friendly businesses
- **Website Content**: Edit company information including mission, goals, and descriptions

## Technology Stack
- **Backend**: PHP with MySQLi database connectivity
- **Frontend**: HTML5, CSS3, and JavaScript
- **Database**: MySQL (configured for local development)
- **Architecture**: MVC-style component structure with reusable PHP templates
